חזרה ללוח הבקרה

🖥️ הגדרת עמדת לקוח

הגדרת מחשב Ubuntu כעמדת צפייה - התקנת תעודות, דפדפן, קיצור דרך ושעון

בחירת מצב הפעלה

מצב סקריפט — יופק קובץ bash שניתן להוריד ולהריץ ישירות על העמדה. מתאים כאשר אין SSH מותקן.

חיבור לעמדת הלקוח

הזן כתובת IP ולחץ על "בדוק חיבור"

הגדרות

💡 יש לוודא שקובץ ה-deb של Chrome נמצא בנתיב שצוין בשרת. הקובץ יועבר לעמדת הלקוח ויותקן אוטומטית.
💡 הסקריפט יוריד את קובץ Chrome ואת תעודת ה-SSL מהשרת. יש לוודא שכתובת השרת נגישה מעמדת הלקוח.

שלבי ההגדרה

🔒
התקנת תעודת SSL
העתקת תעודת SSL למחשב הלקוח והתקנתה כתעודה מהימנה - למניעת אזהרות אבטחה בדפדפן
⏰
הגדרת שעון (NTP)
הגדרת סנכרון שעה עם שרת NTP ברשת המקומית
🌐
התקנת Google Chrome
העתקת קובץ ההתקנה מהשרת והתקנת Chrome על העמדה
🖥️
יצירת קיצור דרך בשולחן העבודה
יצירת אייקון גדול "טל טק" שפותח את Chrome במסך מלא בכתובת CUBE
🔓
ביטול בקשת סיסמה בדפדפן
ביטול חלון ה-Keyring שמבקש סיסמה בעת פתיחת Chrome
✅ הסקריפט הורד בהצלחה!
כעת פתח טרמינל בעמדת הלקוח והרץ את הפקודה הבאה:
sudo bash ~/Downloads/setup-client.sh
💡 הפקודה sudo bash מריצה את הסקריפט ישירות — אין צורך ב-chmod
